Условия для оптовых покупателей

Для оптовых покупателей мы предоставляем индивидуальные условия которые обсуждаются лично с менеджерами по продажам. Для связи с менеджером нужно заполнить форму обратной связи или позвонить по телефонам указанным в разделе контакты: «отдел продаж».

Доставка

Доставка осуществляется в любой регион мира за счет клиента, любой транспортной компанией. Если Вы не знакомы с условиями транспортных компаний наши менеджеры дадут Вам рекомендации по тому, как можно оптимизировать такие расходы.
